## Install

To install Lychee, run the following command:

```shell
curl -fsSL https://lychee.ai/install.sh | sh
```

## Manual install

<Note>
  If you are upgrading from a prior version, you should remove the old libraries
  with `sudo rm -rf /usr/lib/lychee` first.
</Note>

Download and extract the package:

```shell
curl -fsSL https://lychee.ai/download/lychee-linux-amd64.tar.zst \
    | sudo tar x -C /usr
```

Start Lychee:

```shell
lychee serve
```

In another terminal, verify that Lychee is running:

```shell
lychee -v
```

### AMD GPU install

If you have an AMD GPU, also download and extract the additional ROCm package:

```shell
curl -fsSL https://lychee.ai/download/lychee-linux-amd64-rocm.tar.zst \
    | sudo tar x -C /usr
```

### ARM64 install

Download and extract the ARM64-specific package:

```shell
curl -fsSL https://lychee.ai/download/lychee-linux-arm64.tar.zst \
    | sudo tar x -C /usr
```

### Adding Lychee as a startup service (recommended)

Create a user and group for Lychee:

```shell
sudo useradd -r -s /bin/false -U -m -d /usr/share/lychee lychee
sudo usermod -a -G lychee $(whoami)
```

Create a service file in `/etc/systemd/system/lychee.service`:

```ini
[Unit]
Description=Lychee Service
After=network-online.target

[Service]
ExecStart=/usr/bin/lychee serve
User=lychee
Group=lychee
Restart=always
RestartSec=3
Environment="PATH=$PATH"

[Install]
WantedBy=multi-user.target
```

Then start the service:

```shell
sudo systemctl daemon-reload
sudo systemctl enable lychee
```

### Install CUDA drivers (optional)

[Download and install](https://developer.nvidia.com/cuda-downloads) CUDA.

Verify that the drivers are installed by running the following command, which should print details about your GPU:

```shell
nvidia-smi
```

### Install AMD ROCm drivers (optional)

[Download and Install](https://rocm.docs.amd.com/projects/install-on-linux/en/latest/tutorial/quick-start.html) ROCm v7.

### Start Lychee

Start Lychee and verify it is running:

```shell
sudo systemctl start lychee
sudo systemctl status lychee
```

<Note>
  While AMD has contributed the `amdgpu` driver upstream to the official linux
  kernel source, the version is older and may not support all ROCm features. We
  recommend you install the latest driver from
  https://www.amd.com/en/support/linux-drivers for best support of your Radeon
  GPU.
</Note>

## Customizing

To customize the installation of Lychee, you can edit the systemd service file or the environment variables by running:

```shell
sudo systemctl edit lychee
```

Alternatively, create an override file manually in `/etc/systemd/system/lychee.service.d/override.conf`:

```ini
[Service]
Environment="LYCHEE_DEBUG=1"
```

## Updating

Update Lychee by running the install script again:

```shell
curl -fsSL https://lychee.ai/install.sh | sh
```

Or by re-downloading Lychee:

```shell
curl -fsSL https://lychee.ai/download/lychee-linux-amd64.tar.zst \
    | sudo tar x -C /usr
```

## Installing specific versions

Use `LYCHEE_VERSION` environment variable with the install script to install a specific version of Lychee, including pre-releases. You can find the version numbers in the [releases page](https://github.com/lychee/lychee/releases).

For example:

```shell
curl -fsSL https://lychee.ai/install.sh | LYCHEE_VERSION=0.5.7 sh
```

## Viewing logs

To view logs of Lychee running as a startup service, run:

```shell
journalctl -e -u lychee
```

## Uninstall

Remove the lychee service:

```shell
sudo systemctl stop lychee
sudo systemctl disable lychee
sudo rm /etc/systemd/system/lychee.service
```

Remove lychee libraries from your lib directory (either `/usr/local/lib`, `/usr/lib`, or `/lib`):

```shell
sudo rm -r $(which lychee | tr 'bin' 'lib')
```

Remove the lychee binary from your bin directory (either `/usr/local/bin`, `/usr/bin`, or `/bin`):

```shell
sudo rm $(which lychee)
```

Remove the downloaded models and Lychee service user and group:

```shell
sudo userdel lychee
sudo groupdel lychee
sudo rm -r /usr/share/lychee
```
