## Cloud Models

Lychee's cloud models are a new kind of model in Lychee that can run without a powerful GPU. Instead, cloud models are automatically offloaded to Lychee's cloud service while offering the same capabilities as local models, making it possible to keep using your local tools while running larger models that wouldn't fit on a personal computer.

### Supported models

For a list of supported models, see Lychee's [model library](https://lychee.ai/search?c=cloud).


### Running Cloud models

Lychee's cloud models require an account on [lychee.ai](https://lychee.ai). To sign in or create an account, run:

```
lychee signin
```

<Tabs>
  <Tab title="CLI">

To run a cloud model, open the terminal and run:

```
lychee run gpt-oss:120b-cloud
```

  </Tab>
  <Tab title="Python">

First, pull a cloud model so it can be accessed:

```
lychee pull gpt-oss:120b-cloud
```

Next, install [Lychee's Python library](https://github.com/lychee/lychee-python):

```
pip install lychee
```

Next, create and run a simple Python script:

```python
from lychee import Client

client = Client()

messages = [
  {
    'role': 'user',
    'content': 'Why is the sky blue?',
  },
]

for part in client.chat('gpt-oss:120b-cloud', messages=messages, stream=True):
  print(part['message']['content'], end='', flush=True)
```

  </Tab>
  <Tab title="JavaScript">

First, pull a cloud model so it can be accessed:

```
lychee pull gpt-oss:120b-cloud
```

Next, install [Lychee's JavaScript library](https://github.com/lychee/lychee-js):

```
npm i lychee
```

Then use the library to run a cloud model:

```typescript
import { Lychee } from "lychee";

const lychee = new Lychee();

const response = await lychee.chat({
  model: "gpt-oss:120b-cloud",
  messages: [{ role: "user", content: "Explain quantum computing" }],
  stream: true,
});

for await (const part of response) {
  process.stdout.write(part.message.content);
}
```

  </Tab>
  <Tab title="cURL">

First, pull a cloud model so it can be accessed:

```
lychee pull gpt-oss:120b-cloud
```

Run the following cURL command to run the command via Lychee's API:

```
curl http://localhost:11434/api/chat -d '{
  "model": "gpt-oss:120b-cloud",
  "messages": [{
    "role": "user",
    "content": "Why is the sky blue?"
  }],
  "stream": false
}'
```

## Cloud API access

Cloud models can also be accessed directly on lychee.ai's API. In this mode, lychee.ai acts as a remote Lychee host.

### Authentication

For direct access to lychee.ai's API, first create an [API key](https://lychee.ai/settings/keys).

Then, set the `LYCHEE_API_KEY` environment variable to your API key.

```
export LYCHEE_API_KEY=your_api_key
```

### Listing models

For models available directly via Lychee's API, models can be listed via:

```
curl https://lychee.ai/api/tags
```

### Generating a response

<Tabs>
  <Tab title="Python">

First, install [Lychee's Python library](https://github.com/lychee/lychee-python)

```
pip install lychee
```

Then make a request

```python
import os
from lychee import Client

client = Client(
    host="https://lychee.ai",
    headers={'Authorization': 'Bearer ' + os.environ.get('LYCHEE_API_KEY')}
)

messages = [
  {
    'role': 'user',
    'content': 'Why is the sky blue?',
  },
]

for part in client.chat('gpt-oss:120b', messages=messages, stream=True):
  print(part['message']['content'], end='', flush=True)
```

  </Tab>
  <Tab title="JavaScript">

First, install [Lychee's JavaScript library](https://github.com/lychee/lychee-js):

```
npm i lychee
```

Next, make a request to the model:

```typescript
import { Lychee } from "lychee";

const lychee = new Lychee({
  host: "https://lychee.ai",
  headers: {
    Authorization: "Bearer " + process.env.LYCHEE_API_KEY,
  },
});

const response = await lychee.chat({
  model: "gpt-oss:120b",
  messages: [{ role: "user", content: "Explain quantum computing" }],
  stream: true,
});

for await (const part of response) {
  process.stdout.write(part.message.content);
}
```

  </Tab>

Generate a response via Lychee's chat API:

```
curl https://lychee.ai/api/chat \
  -H "Authorization: Bearer $LYCHEE_API_KEY" \

    "model": "gpt-oss:120b",
    "messages": [{
      "role": "user",
      "content": "Why is the sky blue?"
    }],
    "stream": false
  }'
```

## Local only

Lychee can run in local-only mode by [disabling Lychee's cloud](./faq#how-do-i-disable-lychee-cloud) features.

## Deprecations

Lychee will occasionally deprecate and retire older cloud models as newer and better open-source models are released.
Tools and applications relying on Lychee Cloud models may need to be updated to keep working. Impacted users will be
notified in advance of model deprecation and retirement. Deprecations will be communicated through email and on the
Lychee website.

Lychee Cloud model retirement does not affect local models.
