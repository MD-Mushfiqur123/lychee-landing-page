# pip install openai
from openai import OpenAI

client = OpenAI(
  base_url="http://localhost:11434/v1",
  api_key="lychee"  # any string
)

response = client.chat.completions.create(
  model="llama3.1:8b",
  messages=[{"role":"user","content":"Hello!"}]
)
print(response.choices[0].message.content)
// npm install openai
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'http://localhost:11434/v1',
  apiKey: 'lychee'
});

const chat = await client.chat.completions.create({
  model: 'qwen3:32b',
  messages: [{ role: 'user', content: 'Hello!' }],
  stream: true
});
for await (const chunk of chat) process.stdout.write(chunk.choices[0]?.delta?.content ?? '');

# pip install lychee
import lychee

response = lychee.chat(
  model="llama3.1:8b",

      "role": "user",

Exclusive Commands

Tools built for power users

Four commands you won't find anywhere else — available only in Lychee.

⟨⟩

lychee embed

Generate vector embeddings from stdin or arguments. Output as JSON, CSV, or a human-readable preview. Pipe directly into your RAG pipeline.

lychee embed "hello world" --output json
⌗

lychee scan

Detect your GPU, VRAM, and system RAM then get a ranked list of models that will run smoothly on your hardware — no guessing.

lychee scan
⟺

lychee compare

Run two models on the same prompt simultaneously and see outputs + speed + token counts side by side. Pick the right model for each task.

lychee compare llama3.2:3b qwen3:4b
◈

lychee stats

Live dashboard showing all running models — VRAM used, tokens/second, context length, and time until unload. Like htop for your AI.

lychee stats --interval 1
Ecosystem & Community

Built for builders

Discover new models, connect with developers, and seamlessly move your models across environments.

lychee community

Connect with the Lychee open-source community. Fetch links, browse Discord, and automatically pull "good first issues" right from your terminal.

lychee community

lychee serve-catalog

Host your own decentralized model registry. Instantly share your fine-tuned models with your team via a unified HTTP catalog protocol.

lychee serve-catalog --port 8080

lychee generate-client

Stop writing boilerplate. Generate fully-typed, production-ready SDKs for your models in Go, Python, and TypeScript instantly.

lychee generate-client llama3 --lang go

lychee export / import

Package your models into offline `.lychee` tarballs. Perfect for air-gapped deployments, backups, and peer-to-peer sharing.

lychee export my-model backup.lychee
