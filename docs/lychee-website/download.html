Install via Terminal

The fastest way to install Lychee — one command and you're running.

# Install Lychee with one command
curl -fsSL https://lychee.ai/install.sh | sh

# Then run your first model
lychee run llama3.1:8b
# Run in PowerShell as Administrator
Invoke-WebRequest -Uri https://lychee.ai/install.ps1 -OutFile install.ps1
.\install.ps1

# Then run your first model
lychee run llama3.1:8b
# Install via Homebrew (macOS / Linux)
brew tap lychee/tap
brew install lychee

# Then run your first model
lychee run llama3.1:8b

Docker

Run Lychee in a container — perfect for servers, CI, and cloud deployments.

CPU only
docker run -d -v lychee:/root/.lychee -p 11434:11434 --name lychee lychee/lychee
NVIDIA GPU
docker run -d --gpus=all -v lychee:/root/.lychee -p 11434:11434 --name lychee lychee/lychee
Docker Compose
services:
  lychee:
    image: lychee/lychee
    ports: ["11434:11434"]
    volumes: [lychee:/root/.lychee]
    restart: unless-stopped

Integrity

Verify Your Download

All Lychee binaries are signed. When pulling from HuggingFace, lychee hf pull automatically verifies SHA256 checksums.

Verify binary hash (Linux/macOS)
sha256sum lychee-linux-amd64
# Compare with the .sha256 file from the release page
HuggingFace model auto-verification (built-in)
lychee hf pull bartowski/Meta-Llama-3.1-8B-Instruct-GGUF
# SHA256 is automatically verified against HuggingFace LFS checksums
✓ SHA256 verified: Meta-Llama-3.1-8B-Instruct-Q4_K_M.gguf
How to Verify Your Download Checksum

Every Lychee release ships with a .sha256 file. Compare it against your downloaded binary to confirm it hasn't been tampered with.

Linux / macOS
sha256sum -c lychee-linux-amd64.sha256
macOS (shasum)
shasum -a 256 -c lychee-darwin-arm64.sha256
Windows (PowerShell)
(Get-FileHash LycheeSetup.exe -Algorithm SHA256).Hash

A OK or matching hash confirms the file is authentic. GPG signature verification is also available on the releases page.

Build from Source

Want to build Lychee yourself? You'll need Go 1.22+ and CMake 3.24+.

git clone https://github.com/lychee/lychee
cd lychee
go build -o lychee .
# Or with GPU support:
cmake -B build . && cmake --build build
