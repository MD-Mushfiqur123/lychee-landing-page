## System Requirements

* MacOS Sonoma (v14) or newer
* Apple M series (CPU and GPU support) or x86 (CPU only)


## Filesystem Requirements

The preferred method of installation is to mount the `lychee.dmg` and drag-and-drop the Lychee application to the system-wide `Applications` folder.  Upon startup, the Lychee app will verify the `lychee` CLI is present in your PATH, and if not detected, will prompt for permission to create a link in `/usr/local/bin`

Once you've installed Lychee, you'll need additional space for storing the Large Language models, which can be tens to hundreds of GB in size.  If your home directory doesn't have enough space, you can change where the binaries are installed, and where the models are stored.

### Changing Install Location

To install the Lychee application somewhere other than `Applications`, place the Lychee application in the desired location, and ensure the CLI `Lychee.app/Contents/Resources/lychee` or a sym-link to the CLI can be found in your path.  Upon first start decline the "Move to Applications?" request.


## Troubleshooting

Lychee on MacOS stores files in a few different locations.
- `~/.lychee` contains models and configuration
- `~/.lychee/logs` contains logs
    - *app.log* contains most recent logs from the GUI application
    - *server.log* contains the most recent server logs
- `<install location>/Lychee.app/Contents/Resources/lychee` the CLI binary

## Uninstall

To fully remove Lychee from your system, remove the following files and folders:

```
sudo rm -rf /Applications/Lychee.app
sudo rm /usr/local/bin/lychee
rm -rf "~/Library/Application Support/Lychee"
rm -rf "~/Library/Saved Application State/com.electron.lychee.savedState"
rm -rf ~/Library/Caches/com.electron.lychee/
rm -rf ~/Library/Caches/lychee
rm -rf ~/Library/WebKit/com.electron.lychee
rm -rf ~/.lychee
```
