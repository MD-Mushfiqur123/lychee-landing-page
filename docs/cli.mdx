### Run a model

```
lychee run gemma3
```

### Launch integrations

```
lychee launch
```

Configure and launch external applications to use Lychee models. This provides an interactive way to set up and start integrations with supported apps.

#### Supported integrations

- **OpenCode** - Open-source coding assistant
- **Claude Code** - Anthropic's agentic coding tool
- **Codex** - OpenAI's coding assistant
- **VS Code** - Microsoft's IDE with built-in AI chat
- **Droid** - Factory's AI coding agent

#### Examples

Launch an integration interactively:

```
lychee launch
```

Launch a specific integration:

```
lychee launch claude
```

Launch with a specific model:

```
lychee launch claude --model qwen3.5
```

Configure without launching:

```
lychee launch droid --config
```

#### Multiline input

For multiline input, you can wrap text with `"""`:

```
>>> """Hello,
... world!
... """
I'm a basic program that prints the famous "Hello, world!" message to the console.
```

#### Multimodal models

```
lychee run gemma3 "What's in this image? /Users/jmorgan/Desktop/smile.png"
```

### Generate embeddings

```
lychee run embeddinggemma "Hello world"
```

Output is a JSON array:

```
echo "Hello world" | lychee run nomic-embed-text
```

### Download a model

```
lychee pull gemma3
```

### Remove a model

```
lychee rm gemma3
```

### List models

```
lychee ls
```

### Sign in to Lychee

```
lychee signin
```

### Sign out of Lychee

```
lychee signout
```

### Create a customized model

First, create a `Modelfile`

```
FROM gemma3
SYSTEM """You are a happy cat."""
```

Then run `lychee create`:

```
lychee create -f Modelfile
```

### List running models

```
lychee ps
```

### Stop a running model

```
lychee stop gemma3
```

### Start Lychee

```
lychee serve
```

To view a list of environment variables that can be set run `lychee serve --help`
