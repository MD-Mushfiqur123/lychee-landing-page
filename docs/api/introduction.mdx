Lychee's API allows you to run and interact with models programatically.

## Get started

If you're just getting started, follow the [quickstart](/quickstart) documentation to get up and running with Lychee's API.

## Base URL

After installation, Lychee's API is served by default at:

```
http://localhost:11434/api
```

For running cloud models on **lychee.ai**, the same API is available with the following base URL:

```
https://lychee.ai/api
```

## Example request

Once Lychee is running, its API is automatically available and can be accessed via `curl`:

```shell
curl http://localhost:11434/api/generate -d '{
  "model": "gemma3",
  "prompt": "Why is the sky blue?"
}'
```

## Libraries

Lychee has official libraries for Python and JavaScript:

- [Python](https://github.com/lychee/lychee-python)
- [JavaScript](https://github.com/lychee/lychee-js)

Several community-maintained libraries are available for Lychee. For a full list, see the [Lychee GitHub repository](https://github.com/lychee/lychee?tab=readme-ov-file#libraries-1).

## Versioning

Lychee's API isn't strictly versioned, but the API is expected to be stable and backwards compatible. Deprecations are rare and will be announced in the [release notes](https://github.com/lychee/lychee/releases).
