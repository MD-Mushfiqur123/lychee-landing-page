## Install

Install the [Codex CLI](https://developers.openai.com/codex/cli/). For the desktop app, see [Codex App](/integrations/codex-app).

```
npm install -g @openai/codex
```

## Usage with Lychee

<Note>Codex requires a larger context window. It is recommended to use a context window of at least 64k tokens.</Note>

### Quick setup

```
lychee launch codex
```

When launched through `lychee launch codex`, Lychee refreshes the model catalog
and uses a dedicated Codex profile for that session.

To configure without launching:

```shell
lychee launch codex --config
```

To remove the Lychee launch profile and generated model catalog:

```shell
lychee launch codex --restore
```

### Manual setup

To use `codex` with Lychee, use the `--oss` flag:

```
codex --oss
```

To use a specific model, pass the `-m` flag:

```
codex --oss -m gpt-oss:120b
```

To use a cloud model:

```
codex --oss -m gpt-oss:120b-cloud
```

### Profile-based setup

For a persistent Codex CLI configuration, create `~/.codex/lychee-launch.config.toml`:

```toml
model = "gpt-oss:120b"
model_provider = "lychee-launch"
model_catalog_json = "/Users/you/.codex/model.json"

[model_providers.lychee-launch]
name = "Lychee"
base_url = "http://localhost:11434/v1/"
wire_api = "responses"
```

Then run:

```
codex --profile lychee-launch
```
