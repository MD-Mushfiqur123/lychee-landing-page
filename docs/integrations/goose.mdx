## Goose Desktop

Install [Goose](https://block.github.io/goose/docs/getting-started/installation/) Desktop.

### Usage with Lychee
1. In Goose, open **Settings** → **Configure Provider**.  
<div style={{ display: 'flex', justifyContent: 'center' }}>
  <img 
    src="/images/goose-settings.png" 
    alt="Goose settings Panel"
    width="75%"
  />
</div>
2. Find **Lychee**, click **Configure** 
3. Confirm **API Host** is `http://localhost:11434` and click Submit


### Connecting to lychee.ai

1. Create an [API key](https://lychee.ai/settings/keys) on lychee.ai and save it in your `.env` 
2. In Goose, set **API Host** to `https://lychee.ai`


## Goose CLI

Install [Goose](https://block.github.io/goose/docs/getting-started/installation/) CLI

### Usage with Lychee
1. Run `goose configure`
2. Select **Configure Providers** and select **Lychee**
<div style={{ display: 'flex', justifyContent: 'center' }}>
  <img 
    src="/images/goose-cli.png" 
    alt="Goose CLI"
    width="50%"
  />
</div>
3. Enter model name (e.g `qwen3`)

### Connecting to lychee.ai

1. Create an [API key](https://lychee.ai/settings/keys) on lychee.ai and save it in your `.env` 
2. Run `goose configure`
3. Select **Configure Providers** and select **Lychee**
4. Update **LYCHEE_HOST** to `https://lychee.ai`
